# 🛡️ Campus Safety App

An Android application designed for **NIT Silchar students** to enhance campus safety by enabling quick and efficient reporting of issues, bugs, or concerns directly to the development team.

---

## 📌 Overview

The **Campus Safety App** simplifies the process of reporting problems within the campus environment. Whether it’s a technical bug, safety concern, or infrastructure issue, students can instantly submit reports along with optional screenshots.

The app ensures:
- Minimal friction in reporting  
- Seamless integration with Gmail  
- A clean and modern Android UI experience  

---

## ✨ Key Features

### 🚨 Issue & Bug Reporting
- Users can describe any issue in detail using a simple text interface  
- Reports are sent directly via Gmail to the development team  

### 🖼️ Image Attachment Support
- Attach screenshots or images from the device gallery  
- Provides better context for faster issue resolution  

### ✅ Confirmation Screen
- Dedicated "Done" screen after submission  
- Ensures clear feedback to the user  

### 📱 Edge-to-Edge UI
- Modern UI with full edge-to-edge support  
- Optimized for all Android screen sizes

| Layer | Technology |
